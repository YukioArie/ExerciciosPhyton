# Teste e Qualidade de Software

* Algumas Atividades para Exercitar Programação com Testes Unitários Automatizados.

## Orientações

- Efetue um *Fork* do repositório;
- Faça um clone do seu fork `git clone`;
- Escolha a pasta em que gostaria de resolver um exercício;
- Caso deseja contribuir com uma solução em um dos exercícios:
  - Adicione um arquivo com seu nome contendo a resolução do exercício;
- Caso deseja contribuir desenvolvendo um teste em um dos exercícios:
  - Adicione Adicione um arquivo com seu nome contendo a teste do exercício;
- Certifique-se de que seu arquivo está executando sem erros;
- Crie um Pull Request nesse repositório
  - Lembre-se que não é necessário abrir um segundo *Pull Request* se efetuar um commit enquanto seu primeiro *Pull Request* não foi aceito ou fechado.
